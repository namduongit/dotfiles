# Dotfiles - Arch Linux

My personal dotfiles for Arch Linux with DWM (X11).

## Installation

### Clone repository

```bash
git clone https://github.com/namduongit/dotfiles ~/Builds/dotfiles
cd ~/Builds/dotfiles
```

### Run installation script

```bash
chmod +x install.sh
./install.sh
```

The script will automatically:
- Install required packages
- Build and install DWM
- Configure applications (Alacritty, Rofi, etc.)
- Copy configuration files

## Requirements

- Arch Linux x86_64
- Internet connection
